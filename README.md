### Ruby version
Ruby version 2 and above

### System dependencies

#### Configuration
install elasticsearch using
`brew install elasticsearch` on Mac

`go to localhost:9200` to see a json response

#### Database creation
  `rails db:setup` to create and setup the database

#### Database Migration
  `rails db:migrate`

#### Start the rails server
  `rails server`
